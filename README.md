# student-management-api


A simple REST API built using **FastAPI** to manage student records.

## Features 

* Create student
* Get all students
* Get student by ID
* Delete student

## Tech Stack

* Python
* FastAPI
* SQLAlchemy
* SQLite

## Run Project

Install dependencies

pip install -r requirements.txt

Run server

uvicorn main:app --reload

## API Documentation

Open in browser:

http://127.0.0.1:8000/docs
